# C++ Book Management System - Setup Guide

## 项目概述
这是一个用C++编写的图书管理系统，包含以下功能：
- 登录系统（密码：123456）
- 添加图书
- 删除图书
- 修改图书信息
- 查询图书（按ID、书名、作者）
- 浏览图书（分页显示）
- 保存数据到文件

## 文件说明

### 原始文件结构
- `main.cpp` - 主程序入口
- `book.h` - 头文件，定义结构和函数声明
- `add.cpp` - 添加图书功能
- `delete.cpp` - 删除图书功能
- `update.cpp` - 修改图书功能
- `query.cpp` - 查询图书功能
- `functions.cpp` - 辅助函数实现
- `books.txt` - 数据存储文件
