# C++ Book Management System - Setup Guide

## 项目概述
这是一个用C++编写的图书管理系统，包含以下功能：
- 登录系统（密码：123456）
- 添加图书
- 删除图书
- 修改图书信息
- 查询图书（按ID、书名、作者）
- 浏览图书（分页显示）
- 保存数据到文件

## 文件说明

### 原始文件结构
- `main.cpp` - 主程序入口
- `book.h` - 头文件，定义结构和函数声明
- `add.cpp` - 添加图书功能
- `delete.cpp` - 删除图书功能
- `update.cpp` - 修改图书功能
- `query.cpp` - 查询图书功能
- `functions.cpp` - 辅助函数实现
- `books.txt` - 数据存储文件

### 登录
- 默认密码：**123456**
- 支持密码输入隐藏显示（*号）
- 3次错误后系统退出

### 菜单选项
1. **添加图书** - 输入ID、书名、作者、价格、出版日期、出版社
2. **删除图书** - 输入要删除的图书ID
3. **修改图书** - 输入要修改的图书ID，按需修改字段
4. **查询图书** - 支持按ID、书名、作者查询
5. **浏览图书** - 分页显示所有图书（每页5条）
6. **保存数据** - 保存数据到books.txt文件
7. **退出系统** - 退出程序

### 数据持久化
- 程序启动时自动从books.txt加载数据
- 修改后手动选择"保存数据"或程序退出时自动保存

## 编码和国际化说明
- 原始文件使用中文注释，可能在某些编码不兼容的环境出现乱码
- book_system.cpp已转换为英文注释，避免编码问题
- 程序支持中文界面和中文数据输入/输出
